Earthquakes + Spaghetti Towers

Question

How does the shapes used in a structure impact how it holds up during an earthquake? What variables should we test to figure this out?

Round 1

Each person should choose one variable and build a test structure based on that variable

Discussion

What did we learn from these tests? What features seem to strengthen the strucutre? Are there any tradeoffs

Round 2

Based on what we learned, everyone gets a second change to build the tallest structure that can withstand a earthquake.
