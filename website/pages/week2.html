Arduino + Physical Circuits

General Arduino Create Steps

Log into Arduino Create using the account you created last class. Follow the next 6 steps to run the Blink sketch on your Arduino -- this is the "hello world" equivalent on Arduino. Once we do this, we can run the code you made in Tinkercad.

Step 1

After logging into Arduino Create, you should land on a page that looks like this.

Step 2

Connect the power cord to the Elegoo Uno R3 and then plug the other end into one of the USB ports

The Uno should light up one you plug in and there should be a small pop up saying Arduino Uno has been connected

Step 3

Inside your sketch, click on the black bar shown by the red arrow below.

If the first option in the drop-down menu says Arduino Uno at /dev/ttyACM#, click that so a blue check mark appears next to it. Skip to step 5!

If the first option is NOT Arduino Uno at /dev/ttyACM#, click Select Other Board & Port from the dropdown

Step 4

In the window that appears, search for and select Arduino Uno for your board. The ports menu is a little more complicated. Usually the software figures it out for you, but if it doesn't select the first option for now and then go back and change it if the next steps don't work (ask me for help if you're stuck). Click ok to accept the changes.

Step 5

Get ready to run your first sketch! Next to the black bar is a check mark. Click on the check mark to verify your sketch. If everything is good to go, a green Success bar will pop up

Step 6

Final step - press the upload arrow next to the black bar. Again a green Success bar will pop up.

Look down at your Arduino. A yellow light should be flashing on and off every second!

Running Tinkercad Code on Arduino
Step 1

Go back to Tinkercad and open the page where you have your virtual Arduino and Code. Click the drop down titled Block and switch to the Block + Text option. Your screen should look like this.

Step 2

Go to Arduino Create and click new sketch. Go back to the Tinkercad page and copy the code inside the text-box inside Tinkercad. Paste this code inside your new sketch and run the code as we did in the 'General Arduino Setup Section'. Voila! It should work.

Types of Sensors

LEDs are not the only things we can control using our Arduinos. Sensors, as you might guess, are things that can sense or measure things. Today, we will be looking at sensors that can measure light amount, temperature, resistance and distance.

All circuits and code can be find right here. Click me!

Button

Can record when a user presses or releases the button.

Thermistor

Measures the temperature surrounding the sensor. Acts as a rough thermometer.

Photoresistor

Measures the amount of light hitting the sensor. The range of output is from 0 to about 600.

Potentiometer

An adjustable resistor. As you turn the knob on a potentiometer, the resistance changes. Can be used to measure turn distance.

Ultrasonic Sensor

Uses reflecting soundwaves to measure distance, much like a bat uses echolocation. This sensor can measure to about 6 ft of distance but seems to lose accuracy after 5 ft.
